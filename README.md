# Lifelog Pattern Data Generation

라이프로그 패턴 데이터 생성 프로젝트

## 개요

생체신호 센서 및 EMA(Ecological Momentary Assessment) 데이터를 기반으로 라이프로그 패턴 데이터를 생성하는 시스템

## 주요 기능

- 라이프로그 패턴 데이터 생성
- 시계열 데이터 패턴 분석
- 데이터 시각화 및 검증
