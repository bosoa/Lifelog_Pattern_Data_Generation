해석 가이드

Concordance Index:
• C-index > 0.7: 우수한 식별력
• C-index 0.6-0.7: 적절한 식별력
• C-index < 0.6: 낮은 식별력

우도비 검정 (Likelihood Ratio Test):
• p-value < 0.05: 모델이 통계적으로 유의미함 (귀무가설 기각)
• p-value ≥ 0.05: 모델이 통계적으로 유의미하지 않음
• 귀무가설: 모든 회귀 계수가 0 (변수들이 이벤트 발생에 영향 없음)

📈 Kaplan-Meier 생존 곡선

해석

생존 곡선이 급격히 떨어지는 구간은 이벤트 발생이 집중된 시기를 의미합니다. 계층별 곡선의 차이는 각 그룹의 위험도 차이를 나타냅니다.

⚖️ Hazard Ratios (위험비)

해석

• HR > 1: 해당 변수 증가 시 이벤트 발생 위험 증가
• HR = 1: 이벤트 발생 위험에 영향 없음
• HR < 1: 해당 변수 증가 시 이벤트 발생 위험 감소
