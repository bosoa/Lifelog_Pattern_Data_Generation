⏱️ 시계열 데이터 취합 과정

개요

웨어러블 센서는 24시간 연속적으로 생체 신호를 수집합니다. 이러한 고빈도 시계열 데이터를 생존 분석에 활용하기 위해 일정 기간 단위로 집계(aggregation)하는 과정이 필요합니다.

Step 1: 원시 데이터 수집

수집 주기:

  • 심박수: 1분 단위
  • 체온/피부온도: 5분 단위
  • 걸음 수: 실시간 카운팅
  • 산소포화도: 1분 단위

데이터 형식: 타임스탬프 + 센서값

Step 2: 일별 집계

집계 방법:

  • 평균: 심박수, 체온, 피부온도, 산소포화도, HRV
  • 합계: 걸음 수, 스틱 센서
  • 총 시간: 총 수면, REM 수면, 깊은 수면

결측치 처리: 24시간 중 80% 이상 데이터가 있는 경우에만 집계

Step 3: 품질 관리

이상치 탐지:

  • 생리학적 범위 검증 (예: 심박수 40-200 bpm)
  • IQR 기반 극단값 탐지 (1.5×IQR 기준)
  • 센서 오작동 패턴 감지 (동일 값 연속 반복 등)

처리: 이상치는 제거하거나 전후 값의 이동평균으로 보정

Step 4: 시간 윈도우 정의

관측 윈도우:

  • 일별 집계 데이터를 기반으로 관측 구간 설정
  • 활동 수준에 따라 3단계 계층 구분
  • 각 계층별로 차등화된 관측 기간 할당

Step 5: 정신건강 설문과 매칭

동기화:

  • 정신건강 설문 응답 시점 확인
  • 설문 시점 기준 ±3일 내 센서 데이터 평균
  • 설문-센서 쌍 데이터 생성

결과: 각 설문 응답마다 대응하는 센서 특징 벡터 생성

Step 6: 최종 데이터셋 구성

구조:

  • 각 행: 하나의 관측 시점
  • 10개 센서 특징 + 타겟 점수 + 메타데이터
  • Duration 및 Event 변수 생성

검증: 최종 데이터셋의 일관성 및 완전성 확인

🔧 데이터 전처리

1. 결측치 처리

전략:

  • MCAR (Missing Completely At Random): Listwise deletion
  • MAR (Missing At Random): 이동평균 또는 보간법
  • MNAR (Missing Not At Random): 해당 변수 제외
  • 기준: 결측률 30% 이상인 변수는 분석에서 제외
↓

2. 표준화 (Standardization)

방법: Z-score 정규화

  • 각 변수를 평균 0, 표준편차 1로 변환
  • 공식: z = (x - μ) / σ
  • 목적:
    • 서로 다른 단위의 변수들을 동일 스케일로
    • Cox 모델 계수의 비교 가능성 향상
    • 수치적 안정성 확보
↓

3. 다중공선성 검사

지표: VIF (Variance Inflation Factor)

  • VIF > 10: 심각한 다중공선성
  • VIF 5-10: 중등도 다중공선성
  • 조치: 높은 VIF 변수 중 하나 제거 또는 PCA 적용
  • 대안: L2 정규화 (penalizer=0.1) 사용
↓

4. 최종 검증

체크리스트:

  • ✓ 모든 변수가 수치형인지 확인
  • ✓ 무한대(inf) 또는 NaN 값 없음
  • ✓ 충분한 샘플 크기 (이벤트당 최소 10-15 샘플)
  • ✓ Duration > 0 조건 만족
  • ✓ Event 변수가 0 또는 1인지 확인

⏰ 시간적 특성 및 주기성

일주기 리듬 (Circadian Rhythm)

많은 센서 변수들은 24시간 주기의 일주기 리듬을 따릅니다:

  • 체온: 이른 아침 최저, 오후-저녁 최고
  • 심박수: 수면 중 감소, 활동 시 증가
  • 활동: 낮 시간대 집중, 야간 최소

고려사항: 일별 집계를 통해 일주기 변동성을 평균화하여 개인의 전반적인 생리학적 상태를 대표하는 값을 도출합니다.

주간 패턴 (Weekly Pattern)

일부 변수는 요일별 차이를 보일 수 있습니다:

  • 활동량: 평일 vs. 주말 차이
  • 수면: 주말 수면 시간 증가 경향
  • 스트레스: 업무일과 휴식일의 차이

처리: 설문 시점 기준 ±3일 윈도우를 사용하여 단기 변동성을 포착하면서도 요일 효과를 완화합니다.

✅ 데이터 품질 지표

1. 완전성 (Completeness)

  • 센서 가동률: 전체 관측 기간 중 데이터가 기록된 비율
  • 기준: 80% 이상의 가동률을 가진 데이터만 사용
  • 일별 완전성: 각 변수가 하루 24시간 중 얼마나 측정되었는지

2. 정확성 (Accuracy)

  • 센서 캘리브레이션: 주기적인 센서 보정
  • 교차 검증: 다른 센서 데이터와의 일치성 확인
  • 생리학적 타당성: 의학적으로 가능한 범위 내 값인지 검증

3. 일관성 (Consistency)

  • 시계열 연속성: 급격한 값 변화 탐지
  • 센서 간 일관성: 관련 센서 간 논리적 관계 확인
  • 예: 높은 활동량 시 심박수 증가, 체온 상승 예상

4. 적시성 (Timeliness)

  • 동기화: 센서 데이터와 설문 응답의 시간적 근접성
  • 최대 시차: ±3일 이내
  • 이상적: 설문 당일 센서 데이터 사용

🚀 향후 개선 방향

고급 시계열 특징 추출
