📌 센서 데이터 측정 시점 기준

정신건강 설문 응답 시점을 기준으로 ±3일 이내 센서 데이터의 평균값을 사용합니다.

  • 기준 시점: 우울/스트레스 설문 응답일
  • 센서 집계 기간: 설문일 기준 전후 3일 (총 7일 윈도우)

측정 시점 설정 근거

  • ±3일 윈도우 사용 이유:
    • 단일 일의 데이터는 일시적 변동성이 클 수 있음
    • 7일 평균으로 안정적인 생리학적 상태 반영
    • 설문 응답 시점과 시간적으로 가까워 인과관계 추론 가능
  • 일별 평균의 평균 사용:
    • 각 날마다 센서 데이터를 일별로 집계 (평균)
    • 7일간의 일별 평균값을 다시 평균
    • 이상치 및 센서 오류의 영향 최소화

⚠️ 주의사항

  • 측정 기간 중 센서 데이터가 80% 미만으로 수집된 경우 해당 관측치 제외
  • 센서 오작동 또는 이상치가 발견된 날은 집계에서 제외
  • 설문 응답 시점이 정확하지 않은 경우 응답 기록 시간을 기준으로 사용

현재 방식 선택 이유

  • 연구 목적에 부합: 특정 시점의 생리학적 상태와 정신건강 상태의 연관성 분석
  • 데이터 가용성: 각 개인마다 설문 응답 시점이 다르고 빈도가 낮음
  • 해석 용이성: 센서 값이 고정되어 있어 Hazard Ratio 해석이 명확
  • 계산 효율성: 대규모 데이터셋에서도 빠른 학습 가능

📅 Duration (생존시간) 설정

Duration의 의미

생존 분석에서 Duration은 관측 시작부터 이벤트 발생(또는 관측 종료)까지의 시간입니다. 본 연구에서는 활동 수준(Level)에 따라 차등화된 관측 기간을 할당하였습니다.

Activity Level Level 명 Duration 범위 평균 Duration 설명
Level 0 낮은 활동 10-40일 ~25일 초기 관측 기간, 낮은 활동 수준
Level 1 중간 활동 40-70일 ~55일 중기 관측 기간, 중간 활동 수준
Level 2 높은 활동 70-93일 ~81.5일 후기 관측 기간, 높은 활동 수준

⚠️ 주의: Duration은 실제 관측 시간이 아님

현재 분석에서 사용된 Duration은 시뮬레이션된 값입니다. 활동 수준에 따라 균등 분포로 랜덤하게 할당되었으며, 실제 종단 추적 기간을 나타내지 않습니다.

향후 개선 방향:

  • 실제 설문 응답 간격을 Duration으로 사용
  • 다회차 설문 데이터로 진정한 종단 분석 수행
  • 개인별 실제 추적 기간 반영

🎯 Event (종단점) 정의

✓ 이벤트 발생 기준

  • 우울(Depression): Depression Score ≥ 6점
  • 스트레스(Stress): Stress Score ≥ 5점

임계값 이상의 점수는 임상적으로 의미있는 증상 수준을 나타내며, 조기 개입이 필요한 시점으로 정의됩니다.

Event vs Censored

  • Event = 1 (이벤트 발생):
    • 설문 시점에 임계값 이상의 점수 측정
    • Cox 모델에서 이 시점을 이벤트 발생 시점으로 간주
    • 해당 관측치의 센서 값이 이벤트와 연관된 특징으로 학습됨
  • Event = 0 (Censored, 미발생):
    • 설문 시점에 임계값 미만의 점수
    • 관측 종료 시점까지 이벤트 미발생
    • Right-censored 데이터로 처리

🔧 데이터 전처리

1. 표준화 (Z-score Normalization)

적용 방법

scaler = StandardScaler()
X_standardized = scaler.fit_transform(X)

# 각 변수를 평균 0, 표준편차 1로 변환
# z = (x - μ) / σ

적용 시점: Cox PH 모델 학습 직전

목적:

  • 서로 다른 단위의 센서 변수들을 동일 스케일로 변환
  • Hazard Ratio의 비교 가능성 향상
  • 수치적 안정성 확보

2. 결측치 처리

전략

  • Listwise Deletion: 센서 변수 중 하나라도 결측인 행 제거
  • 기준: 결측률 30% 이상 변수는 분석에서 제외
  • 보정: 일시적 센서 오류는 이동평균으로 보간

3. 이상치 탐지

방법

  • 생리학적 범위 검증: 각 센서의 의학적 정상 범위 확인
  • IQR 기반 탐지: Q1 - 1.5×IQR ~ Q3 + 1.5×IQR 범위 벗어난 값
  • 센서 오작동 패턴: 동일 값 연속 반복, 급격한 변화 등

🚀 향후 개선 방향

1. Time-Varying Covariates 도입

  • 센서 데이터의 시간적 변화를 Cox 모델에 반영
  • 실시간 위험도 업데이트 가능
  • 동적 예측 모델 구축

2. 실제 종단 데이터 활용

  • 다회차 설문 데이터로 진정한 종단 분석
  • 실제 관측 기간을 Duration으로 사용
  • 개인별 변화 패턴 포착

3. 고급 시계열 특징 추출

  • 변동성 지표: 표준편차, 변이계수
  • 추세 특징: 기울기, 변화율
  • 주기성 분석: FFT, autocorrelation
